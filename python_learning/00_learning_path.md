# Python 快速学习路径

## 阶段一：Python基础 (1-2天)

### 1. 变量与数据类型
- 变量定义
- 基本数据类型：int, float, str, bool
- 类型转换
- f-string 格式化

### 2. 运算符
- 算术运算符：+, -, *, /, //, %, **
- 比较运算符：==, !=, <, >, <=, >=
- 逻辑运算符：and, or, not
- 赋值运算符：=, +=, -= 等

### 3. 控制流
- if/elif/else 条件语句
- for 循环
- while 循环
- break 和 continue
- pass 语句

## 阶段二：数据结构 (1天)

### 1. 列表 (List)
- 创建和访问
- 常用方法：append, insert, remove, pop
- 切片操作
- 列表推导式

### 2. 字典 (Dict)
- 创建和访问
- 常用方法：keys(), values(), items()
- 字典推导式

### 3. 元组 (Tuple) 和 集合 (Set)
- 元组的不可变性
- 集合的去重特性

## 阶段三：函数 (1天)

### 1. 函数基础
- 函数定义
- 参数：位置参数、关键字参数、默认参数
- return 语句
- *args 和 **kwargs

### 2. 高级特性
- Lambda 表达式
- 内置函数：map, filter, reduce
- 装饰器基础

## 阶段四：面向对象 (1-2天)

### 1. 类与对象
- 类的定义
- __init__ 方法
- 实例属性和方法
- 类属性和类方法

### 2. 面向对象特性
- 继承
- 多态
- 封装
- 魔术方法：__str__, __repr__ 等

## 阶段五：文件操作与模块 (1天)

### 1. 文件操作
- 读写文本文件
- with 语句
- JSON 处理

### 2. 模块与包
- import 语句
- 标准库介绍
- pip 安装第三方库

## 阶段六：实战项目 (2-3天)

### 项目建议
1. 待办事项管理器
2. 简易计算器
3. 数据分析脚本
4. Web爬虫
5. 小游戏

## 学习建议

1. **动手实践**：每个知识点都要自己写代码验证
2. **逐步深入**：先掌握基础，再学习高级特性
3. **调试技巧**：学会使用 print 和 debugger
4. **阅读文档**：遇到问题查看官方文档
5. **做项目**：理论结合实践，做小项目巩固
